# 🚀 DevOps Major Project: Cloud-Native Application

## 📌 Overview

This project demonstrates a complete DevOps workflow by building, containerizing, and deploying a web application using modern tools and cloud services.

---

## 🛠️ Tech Stack

* Linux (Ubuntu)
* Python (Flask)
* Docker
* Docker Compose
* Nginx (Reverse Proxy)
* AWS ECR (Elastic Container Registry)
* Git & GitHub

---

## ⚙️ Features

* Containerized Flask web application
* Multi-container architecture using Docker Compose
* Nginx reverse proxy for production-like setup
* Cloud image storage using AWS ECR
* Version control using Git

---

## 🚀 How to Run Locally

```bash
docker-compose up -d
```

Then open:
http://localhost

---

## ☁️ Cloud Integration

* Docker image pushed to AWS ECR
* Ready for deployment on cloud servers
